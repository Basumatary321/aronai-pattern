<h1>Aronai Pattern Maker 🎨</h1>

A modern and interactive Aronai Pattern Design Web App built using HTML, CSS, and JavaScript.
Users can customize traditional Aronai pattern colors, zoom, rotate, preview designs, and download in multiple formats like PNG, JPG, PDF, and SVG.

✨ Features
🎨 Change background and pattern colors
🔍 Zoom in / Zoom out
🔄 Rotate design
📱 Mobile responsive UI
🤏 Smooth pinch zoom support on mobile
📥 Export in multiple formats:
Transparent PNG
JPG
PDF
SVG
⚡ Smooth animated download progress popup
🖼️ High-quality SVG rendering
🚀 Lightweight and fast
🛠️ Technologies Used
HTML5
CSS3
JavaScript
SVG Graphics
jsPDF Library
